The given Notebook(ipynb) consists of following Python Libraries:
Tensorflow 2.x CPU Version
os
pandas
nump 
matplotlib
sklearn
keras
librosa


Eager Exceution of Tensorflow must be turned off, by code given in cell 1. 


The dataset can be downloaded using the following Link:
http://download.tensorflow.org/data/speech_commands_v0.02.tar.gz

Please update the dataset folder as target file in unzipped form in 2nd code block under directory variable.

Next, all of the code blocks can be run in jupyter environment.


Please Note that execution of loading the dataset and preprocess may take additional time.
